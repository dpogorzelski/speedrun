# Nyx

Nyx is a small utility to exectue commands remotely at scale

## Installation

Download the precompiled binary from here

```bash
curl https://github.com/dawidpogorzelski/nyx/releases/0.1.0/release
mv nyx /usr/local/bin
```

## Usage

```bash
nyx init
nyx run "whoami"
```
